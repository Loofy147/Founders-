## Get Started

### Web Interface
```bash
# Clone the repository
git clone https://github.com/yourusername/autodeploy-engine.git
cd autodeploy-engine

# Install dependencies
npm install

# Run the dev server
npm run dev

# Visit http://localhost:3000
```

### CLI Tool
```bash
# Install the CLI globally
npm install -g autodeploy-engine

# Navigate to your project directory
cd /path/to/your/project

# Run the interactive setup
autodeploy init
```

## Features

- **Interactive Wizard**: A simple, 4-question setup process to get you started in minutes.
- **Multi-Platform Support**: Generate configurations for over 10 platforms, including Vercel, Netlify, AWS, and Azure.
- **Web UI & CLI**: Choose between a user-friendly web interface and a powerful command-line tool.
- **Production-Ready Templates**: Start with battle-tested configurations that are ready for production.
- **Free Options**: Find the best free-tier options for your project.
- **Auto-Detection**: The CLI automatically detects your project's framework.

## Supported Platforms

- Vercel (Next.js, React)
- Netlify (Static sites, SPAs)
- AWS Amplify (Next.js, React)
- Azure Static Web Apps (Next.js, React)
- GitHub Actions (Android, CI/CD)
- Railway (Backend, Full-stack)
- Docker (Any platform)
